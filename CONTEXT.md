# Context

## Domain Terms

- **Telegram inbox**: private notes or photo-backed inputs captured from the operator's Telegram chat.
- **Idea inventory**: the set of fresh or reusable idea candidates available for a drafting cycle.
- **Draft batch**: a set of up to three post options prepared for Telegram review in one cycle.
- **Review lifecycle**: the operator approval flow over a draft batch, including approve, reject, edit, regenerate, and skip actions.
- **Publication**: a queued, published, or failed outbound X post derived from an approved draft option.
- **Weekly digest**: the operator-facing Telegram message combining engagement suggestions, follow suggestions, and a weekly summary.
- **Companion state**: the filesystem-backed private state repo that stores inbox items, drafts, approvals, publications, summaries, outbox messages, and runtime offsets.
